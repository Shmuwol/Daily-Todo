# Daily-Todo

### To install this app
1. download [this](https://github.com/Shmuwol/Daily-Todo/archive/master.zip) file
2. un-zip the file
3. `cd` to app directory
4. run `bundle install`
5. run `shotgun`
6. go to [http://127.0.0.1:9393](http://127.0.0.1:9393) in the browser
7. get productive.
